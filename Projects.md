## AI Personas for Moms

This project explores how mothers think about and use AI tools. Using survey responses, I analyzed patterns in language, concerns, and use cases to develop user personas.

**Problem:**  
Mothers are using AI in different ways, but their needs, comfort levels, and concerns are not always clearly understood.

**What I Did:**  
- Cleaned and prepared survey response data
- Explored text responses for common themes
- Used clustering to identify persona groups
- Interpreted results into practical user insights

**Tools Used:**  
Python, Pandas, Scikit-learn, Jupyter Notebook, NLP, Clustering

**Status:** In Progress

## Employee Data Cleaning Project

This project focused on identifying and correcting data quality issues in an employee dataset.

**Problem:**  
The dataset contained duplicate records, missing values, invalid salary values, and inconsistent fields that needed to be addressed before analysis.

**What I Did:**  
- Removed duplicate records
- Filled missing numeric values using the median
- Corrected invalid salary values
- Standardized inconsistent values
- Created a cleaned dataset for analysis

**Tools Used:**  
Python, Pandas, Data Cleaning, Data Validation

**Status:** Completed
