# About Me

Hi, I’m **Kamri Williams** — a data-focused technology professional with experience across fintech, enterprise support, analytics, and systems operations.

I currently work as an **Application Support Engineer at Q2 Software**, where I investigate production issues for financial institutions, analyze system behavior, validate data, and translate technical findings into clear business insights.

My background includes experience with:

- Python
- SQL
- Power BI
- Machine learning
- Data visualization
- ETL concepts
- APIs
- Log analysis
- Root cause analysis
- Technical documentation

Before Q2, I worked as a **Project Engineer with Wipro supporting Meta Reality Labs and Facebook Messenger**, where I contributed to testing, validation, debugging, and process improvement for large-scale platforms.

I’m passionate about using data to solve real problems, improve systems, and create better user experiences. My goal is to continue growing in data analytics, analytics engineering, and data science.

Outside of tech, I’m also the founder of **Melanated Mamas**, a nonprofit organization supporting and empowering Black mothers throughout the Golden Crescent region.

## Current Focus

I’m currently focused on:

- Building data science portfolio projects
- Strengthening my machine learning skills
- Applying analytics to real-world business problems
- Growing into data science and analytics engineering roles
