# Employee Data Cleaning Project

This project focused on identifying and correcting data quality issues in an employee dataset.

**Problem:**  
The dataset contained duplicate records, missing values, invalid salary values, and inconsistent fields that needed to be addressed before analysis.

**What I Did:**  
- Removed duplicate records
- Filled missing numeric values using the median
- Corrected invalid salary values
- Standardized inconsistent values
- Created a cleaned dataset for analysis

**Tools Used:**  
Python, Pandas, Data Cleaning, Data Validation

**Status:** Completed
