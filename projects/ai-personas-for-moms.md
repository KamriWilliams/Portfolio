# AI Personas for Moms

This project explores how mothers think about and use AI tools. Using survey responses, I analyzed patterns in language, concerns, and use cases to develop user personas.

**Problem:**  
Mothers are using AI in different ways, but their needs, comfort levels, and concerns are not always clearly understood.

**What I Did:**  
- Cleaned and prepared survey response data
- Explored text responses for common themes
- Used clustering to identify persona groups
- Interpreted results into practical user insights

**Tools Used:**  
Python, Pandas, Scikit-learn, Jupyter Notebook, NLP, Clustering

**Status:** In Progress
