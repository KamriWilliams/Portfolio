# Heart Disease Prediction

This project uses machine learning to predict heart disease risk based on patient health data.

**Problem:**  
Healthcare data can be used to identify risk patterns and support early decision-making.

**What I Did:**  
- Cleaned and prepared health data
- Explored relationships between variables
- Built a classification model
- Evaluated model performance using accuracy and confusion matrix results

**Tools Used:**  
Python, Pandas, Scikit-learn, Logistic Regression, Data Visualization

**Status:** Completed
